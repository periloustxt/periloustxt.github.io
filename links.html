link garden

tool kit

libgen.is
I simply cannot exist without this online library of pirated books. Find almost anything here.

The Anarchist Library
One of my favorite libraries online. Tons of subject categories. Lots of great leftist readings.

textz
A list of wonderful reads related to political futures, art, and technology. View the source code for secret links!

68k.news
Lo-fi HTML feed of popular news headlines. Loads fast, uses little data. Read articles without web bloat.

Raindrop
Very handy web bookmarker/read later service. Browser extension available. Hashtagging, folder organizing, and highlighting of web content included. This has been one of my favorite recent web apps.

inoreader
Since the downfall of Google Reader, there have been few options outside of Feedly for an adequate online RSS reader. Inoreader has a very good, minimalist and customizable interface. I love the plain list view.

Lagrange Gemini Client
A very good Gemini protocol browser. Access the Gemini small web.

SimpleNote
Still a great online note taking service. I use it for quick notes to sync from my phone to my web browsers. From there, I transfer notes over to my Obsidian vault.

Obsidian
I've tried Logseq, but like Obsidian more. Faster, better file organization, and my vault keeps all my notes and PDF's organized there.

SyncThing
Humming along in the background of both my MacOS laptop and my old iMac running Linux Mint XFCE is the very powerful folder synchronization tool SyncThing. I have one folder that houses my Obsidian vault, making it accessible from both computers across different operating systems.

VS Codium
Open source binaries of VS Code, without the Microsoft telemetry. Pretty much the same experience as VS Code, but faster, more private.

Marginalia Search
I love this search engine for parsing out otherwise unseen corners of the web. A great alternative to your regular cast of Google, DuckDuckGo, or Bing.

webwormhole
Send large files through a web wormhole. Both users must be online at the same time. Unique code word generated for each transfer.

Power Thesaurus
A better thesaurus.

addy.io
Make an anonymous e-mail alias and keep spam under control.

LibreOffice
Your complete Microsoft Suite replacement. Word documents, spreadsheets, presentations, everything you'd need. I especially love customizing all the toolbars to make each application a lot less cluttered than Microsoft Word or Apple Pages.

Linux Mint XFCE
I had a very slow-running 2015 iMac that was given a new life with Linux Mint XFCE. Runs very fast on old hardware. I'm convinced I'll never need a Windows or MacOS machine again except maybe to run Logic Pro.
